### JPA SHOP(쇼핑몰 프로젝트)

- 쇼핑몰 프로젝트를 스프링 웹으로 제작


- 사용된 기술 
  - Spring Boot
  - Spring Data Jpa
  - MySql
  - H2 Database
  - Swagger
  - Spring Security
